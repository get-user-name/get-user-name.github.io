Unity와 Mapbox를 이용한 GPS-based AR 앱

May 11, 2019

Unity와 Mapbox를 사용하여 위치기반 AR 앱을 만들어보자.

1. Mapbox

우선, mapbox SDK를 사용하기 위해 계정을 만들고, Unity용 SDK를 다운받는다.

Assets > Import Package > Custom Package
Unity에 새 프로젝트를 만든 후, 다운받은 SDK를 import한다.
Mapbox 메뉴가 생겼을 것이다.

File > Build Settings
에서 Android로 Switch Platorm 한다.
