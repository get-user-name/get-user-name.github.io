Unity에서 png 이미지 사용법

June 4, 2019

투명 배경이 있는 png 이미지를 사용해보자.

  • png를 asset 폴더 안으로 가져온다.
  • Create - Material 을 생성한 후, Inspector 창에서 Albedo에 png 이미지를 넣는다.
  • 생성한 Material의 Shader를 Sprites/Default로 설정한다.
  • Game Object를 생성한 후, Inspector 창에서 Meterial에 2,3에서 생성한 Material을 넣는다.
